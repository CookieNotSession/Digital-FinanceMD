比特幣(一)
---
- 定義: 為世界上第一種基於P2P分佈技術在網路發行和交易的虛擬貨幣
- 由中本聰於2008年發明
- [x] **專屬所有權**
    - 操控你的比特幣需要private key，可以被隔離保存在任何存儲介質
- [x] **低交易費且無隱藏成本**
    - 目前可以免費匯出比特幣，且沒有繁瑣的額度與手續限制，支付時只需要對方比特幣地址
- [x] **去中心化**
    - 以分散式點對點計算的網路設計，無需央行、票據交換所及其他金融機關監管，使用者可以獨立運作
- [x] **匿名設計**
    - P2P的特性係交易紀錄皆為公開 (**因為比特幣匿名，無法推得真正擁有人是誰，所以可以公開**)
    - 為何要公開 ? 因為若沒有公開，**就無法互相比對帳戶是否正確**
    - 採用共識決，每個人都來check此帳戶是否有問題，如果大多通過，就沒問題
    - 因為紀錄皆為公開，可以查到每個帳戶金額及相關交易紀錄，惟無法確認擁有者(無法追蹤實際擁有者)


比特幣(二)
---

- [x] **私人貨幣**
    - 由「礦工」透過「採礦」過程所獲得之酬勞，即為私人鑄幣
- [x] **總量限制 (物以稀為貴)** ==經濟學概念==
    - 比特幣係依前項採礦活動產生，惟系統控制每隔4年貨幣發行數減半。最初4年生產1,050萬個，第2個4年生產525萬個，依次遞減，預計2133年總共生產2,100萬個

比特幣的運作核心: 區塊鏈技術
---
- **早期電子貨幣的問題**
    - 貨幣真偽: 虛擬貨幣容易偽造、竄改
        - 容易竄改是因為早期不太相信帳簿的正確性
    - 重複利用:無法保證虛擬貨幣是否同時支付到不同使用者手中
    
    ```
    書借給你 v.s 電子書借給你
    書借給你，我就看不到了，但電子書我還有
    ```

    - 不透明的交易流程:網路帳戶虛名化、無法提供即時的金錢流向
    - 可能成為洗錢的管道，虛假交易、詐騙、實現資金非法轉移套現
    
- **區塊鏈—建立信任感的中樞系統**
    - 由一群**彼此陌生的**互相連續使用者**共同分擔維護記錄**的工作
    - 給予履行責任的**獎勵**
        - 發比特幣
        - 記帳: 礦工的概念
        - 分散式帳本一群人要大量合作，如何大量合作? 就是要為自己利益而來
        - 利用你的計算能力去取得記帳與驗證的權力
    - 確立正確的共同總帳管理方式 (不會被隨意竄改)
    - 建立信任感

**區塊鏈剖析:**
```
區塊: 記帳的區塊，表示寫了不能改(Block)
鏈: 分佈在各地的區塊，叫做鏈。
```

區塊鏈技術特性
---
其實分散式帳本本身是沒有效率的

1. 去中心化作為分散式節點
由每個參與其中的礦工提供運算資源協助記錄帳本
2. 不可否認
每個交易動作執行時間都會被依序記錄下來，在往後的變更中以利對照
3. 難以竄改
不斷延長的區塊鏈中所添加的每筆交易都會對照現有的總帳進行查核，以大數法則防止某部分獲利
```
大量陌生人，難以共謀 - 社群概念
不可竄改性 - 密碼學概念
```
4. 獎勵
提供紀錄區塊鏈運算資源的「礦工」

![](https://i.imgur.com/RjJx4Fa.jpg)

以上面圖片例子: 需要挖礦中選出**計算能力最好的五個**，其他不錄取，來驗證此Block，並給予比特幣。

- 在銀行領錢，需要比對這個領錢的人是否為本人(驗身分證等動作)
  在區塊鏈世界，透過礦工來取代銀行員驗證的事情

### 金融領域
- 股票市場交易清算
    - 原需費時三天，使用區塊鏈只需十分鐘 **(人工與數位的速度落差)**
- 無須透過交易所中介，直接發行有價證券
    - 美國網購業者Overstock發行5億證券，節省8成以上的管理與中介成本
- 比特幣交易、匯款、股票交易、有價證券發行、相關股票交易將無須經由證券公司、信託公司、與證券保管機構即可進行業務
    - 不用中央背書，如新台幣背書就是台灣政府
- 未來可能廣泛應用於: **匯款、支付、清算、群眾募資、微貸、證券發行管理紀錄、遺囑信託、資產管理、貿易保險**
